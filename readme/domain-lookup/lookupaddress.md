LookupAddress(domain: string, coinName: string): Promise\<undefined | string>

*   **Parameters**

    * **domain: string**
    * **coinName: string (**[**The cryptocurrency name**](../../cryptocurrency-short-name.md)**)**

    **Returns Promise\<undefined | string>**
